Detailed Description

Functions for reading and writing configuration stored in EEPROM.

Macro Definition Documentation

#define DISABLE_EEPROM_CONFIG

Enable or disable EEPROM data configuration. If this macro is defined program do not read nor write data to EEPROM. Default values are used instead.

Function Documentation

void ReadConfig ( )
inline

Reads configuration data from EEPROM. Checks CRC of config data in EEPROM. Validates CRC and use default values in case of damaged data.

Note
If macro DISABLE_EEPROM_CONFIG is defined default parameters are used (not read from flash).
void WriteConfig ( )

Store configuration data in EEPROM. CRC is computed.

Note
If macro DISABLE_EEPROM_CONFIG is defined function does nothing.

Variable Documentation

CONFIGURATION EEMEM ee_Config

Configuration in EEPROM

uint8_t EEMEM ee_ConfigCRC =0xAB

CRC of the configuration data in EEPROM. Checksum is computed only from ee_Config. Startup message does not count.

uint8_t EEMEM ee_Message[64] = " AQUATICUS.INFO"

Start-up message on display. Modify to whatever you like.
