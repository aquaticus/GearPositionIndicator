ISR ( TIMER0_OVF_vect  )

Display interrupt.

The interrupt routine is called every time timer0 overflows. First data from DisplayBuffer is copied to HardwareBuffer, then columns and rows are set.

At time only 8 pixels (row) are on.

Automatic brightness is handles here. There are 16 levels of brightness.

volatile uint8_t DisplayBuffer[8]

Frame buffer.

Data is stored in simple pixel format, rows and columns are in proper order. Copy data to be displayed here.

volatile uint8_t g_LedBrightness = BRIGHTNESS_MAX

LED display brightness. From 0 (dark) to 15 (bright).

char g_TextBuffer[TEXTBUFFER_SIZE]

Buffer used to format text.

Data length is stored in g_TextBufferLen.

uint8_t g_TextBufferLen = 0

Length of the data in g_TextBuffer.

volatile uint8_t HardwareBuffer[8]

Data to be displayed in hardware compatible format.

Do not copy data to this table. Data is copied from DisplayBuffer after displaying one symbol on LED (prevents blinking).

Note
This buffer is used by interrupt, never copy any data directly to this buffer.
