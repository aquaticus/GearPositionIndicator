uint8_t GetGear ( )

Demo gear box procedure.

This is example gear box routine. It does not read real gear box but continuously changes gear every 1s. You must write your own procedure that reads appropriate data from your gear box interface. Typically it reads ADC converter pins.

Warning
This is an example procedure. It does not return real gear number.
Returns
Gear number. 0 for neutral, 1 to 5 for gears.
