Function Documentation

void DisplayTemperature ( )

Displays current temperature on LED.

Scrolling speed have to be the same as in TemperatureLoop() function. In general, it should be impossible to see the difference when temperature is shown by both functions.

int FormatTemperature ( )

Format temperature into string.

Result stored in g_TextBuffer. Length of text is set g_TextBufferLen. String is formatted based on configuration settings.

Returns
Length of the formatted string without terminating 0.

g_Config
uint8_t GetTempConversionResult ( )

Get temperature from DS18B20 sensor.

As result g_nTemperature is set. Temperature conversion must be initiated by StartTempConversion(). Function can read temperature 750ms after calling StartTempConversion();

Warning
Function disable and enable interrupts.
Return values
0success.
1reset error
2CRC error

int16_t Raw2Temp ( uint8_t *  raw)
inline

Convert raw DS18B20 data to temperature integer. Function works properly only for 12 bit result.

Parameters
rawPointer to raw data (2 first bytes) of 12 bit reading.
Returns
Temperature * 10 in Celsius degrees, e.g. -10.6C = -106
uint8_t StartTempConversion ( )

Start DS18B20 temperature conversion.

It takes 750ms to convert temperature. Conversion result must be obtained using GetTempConversionResult().

Warning
Function disable and enable interrupts.
Returns
0 for success.

Variable Documentation

int16_t g_nTemperature = INVALID_TEMP

Current temperature in Celsius * 10. Updated every time by GetTempConversionResult() It can be INVALID_TEMP in case of reading problems.
