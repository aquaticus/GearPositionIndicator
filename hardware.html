Hardware

Hardware configuration

Microcontroller
  • Chip: ATmega 88/168/328 (88p/168p/328p)
  • System clock: 8MHz (Set fuse bits to use internal RC oscillator)
External components (in reference schematic)
  • PC0 - auxiliary pin #1
  • PC1 - 5V ADC input
  • PC2 - button
  • PC3 - 12V ADC input
  • PC4 - light sensor
  • PC5 - 1wire interface for DS18B20 temperature sensor
  • PC6 - auxiliary pin #2
  • PC7 - auxiliary pin #3
LED columns
  • PD0 (OUT) - LED column 6
  • PD1 (OUT) - LED column 3
  • PD2 (OUT) - LED column 1
  • PD3 (OUT) - LED column 2
  • PD4 (OUT) - LED column 4
  • PD5 (OUT) - LED column 8
  • PD6 (OUT) - LED column 5
  • PD7 (OUT) - LED column 7
LED rows
  • PB0 (IN) - LED row 1
  • PB1 (IN) - LED row 6
  • PB2 (IN) - LED row 4
  • PB3 (IN) - LED row 2
  • PB4 (IN) - LED row 3
  • PB5 (IN) - LED row 5
  • PB6 (IN) - LED row 8
  • PB7 (IN) - LED row 7
Notes
Enable pull-up resistors for PC4, PC5, PC2.
PB6 and PB7 configured as I/O lines.
